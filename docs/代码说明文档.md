## 1. 项目概述

SpeakUp AI 是一个 AI 驱动的实时英语口语陪练平台，支持三大场景：

| 场景 | Scene Key | 状态 |
|------|-----------|------|
| 🎯 求职面试 | `interview` | 已启用 |
| 🍽️ 餐厅点餐 | `restaurant` | 已启用 |
| 📊 商务会议 | `meeting` | 已启用 |

### 核心功能

- **实时语音交互** — WebSocket 双向音频流
- **多场景角色扮演** — 面试官 / 服务员 / 会议主持等多种 AI 角色
- **轻量实时纠错** — 语法、发音、表达即时反馈
- **VAR 时间线回放** — 会话事件时间轴，点击跳转对应音频
- **多维度评分报告** — 发音/语法/流利度/逻辑等多维评估
- **简历/JD 上传解析** — 面试场景个性化出题

### 数据流说明

1. **页面加载**: 前端 SSR → 后端 `GET /api/scenes` → 返回场景列表
2. **创建会话**: 前端 → 后端 `POST /api/interviews` → 写入 PostgreSQL → 返回 sessionId + 1 WebSocket URL
3. **实时对话**: 前端 → WebSocket → 后端 → DeepSeek LLM → 后端 → WebSocket → 前端
4. **缓存**: 所有读多写少的数据优先走 Redis，Redis 不可用时自动降级为内存缓存

### 5.2 数据库层 (database.py)

#### 核心函数

```python
def _resolve_url(raw: str) -> str
```
自动为数据库 URL 添加异步驱动前缀：
- `postgresql://` → `postgresql+asyncpg://`
- `sqlite:///` → `sqlite+aiosqlite:///`

```python
async def get_db() -> AsyncSession
```
FastAPI 依赖注入，每次请求提供独立的事务会话。成功自动 commit，异常自动 rollback。

```python
async def init_db()
```
启动时执行：
1. 建表 (`Base.metadata.create_all`)
2. 种子 demo 用户 (`demo@offergpt.local`)

#### 连接池配置

| 数据库 | pool_size | max_overflow |
|--------|-----------|--------------|
| PostgreSQL | 10 | 20 |
| SQLite | 无（不支持） | 无 |

### 5.3 数据模型 (models/base.py)

8 张表，全部使用 **UUID 主键** 和 **UTC 时区时间戳**。JSON 字段在 PostgreSQL 上自动映射为 `jsonb`。

#### 模型关系图

```
User (1) ──┬── (N) Resume
            ├── (N) Job
            ├── (N) ScenePreset
            └── (N) Interview (1) ──┬── (N) TimelineEvent
                                    ├── (1) Report
                                    └── (N) AgentLog
```

#### 模型清单

| 模型 | 表名 | 用途 | 关键字段 |
|------|------|------|----------|
| `User` | users | 用户账号 | email(唯一), name, plan |
| `Resume` | resumes | 简历上传解析 | file_url, parsed_profile(JSON), parse_status |
| `Job` | jobs | JD 上传解析 | title, jd_text, parsed_profile(JSON) |
| `ScenePreset` | scene_presets | 用户场景偏好 | scene, scene_config(JSON) |
| `Interview` | interviews | 训练会话 | scene, topic, transcript(JSON), scene_config(JSON) |
| `TimelineEvent` | timeline_events | 时间线事件 | event_type, severity, start_ms, end_ms, evidence(JSON) |
| `Report` | reports | 评分报告 | dimension_scores(JSON), report_json(JSON) |
| `AgentLog` | agent_logs | Agent 调用日志 | agent_name, input_summary(JSON), latency_ms, tokens |
